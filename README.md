# Full Stack open 2025


The course introduces modern web application development using JavaScript. The main focus is on single page applications implemented using the React library, and the REST interfaces implemented using Node.js that support them. The course also introduces GraphQL, an alternative interface implementation principle to REST.
The course also covers application testing, configuration, and management of execution environments, as well as the use of databases to store application data.
